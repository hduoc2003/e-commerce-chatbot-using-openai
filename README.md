Request có dạng
```json
{
    "conversation": [user_msg, bot_msg, user_msg, bot_msg, ...]
}
```

```json
{
    "conversation": [
        "còn bao nhiêu Áo Khoác Phao Nam size L",
        "Hiện tại còn 43 chiếc Áo Khoác Phao Nam size L đang có sẵn. Nếu bạn quan tâm đến sản phẩm này hoặc có bất kỳ câu hỏi nào khác, đừng ngần ngại để lại tin nhắn cho Lmao ShopBot nhé! Chúng tôi luôn sẵn lòng hỗ trợ bạn.",
```
